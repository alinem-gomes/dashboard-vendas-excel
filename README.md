# Dashboard de Vendas em Excel

![Dashboard](dashboard_vendas.png)

## Objetivo

Projeto desenvolvido para consolidar e analisar indicadores de vendas utilizando Microsoft Excel.

## Funcionalidades

* Dashboard interativo
* Tabelas Dinâmicas
* Gráficos Dinâmicos
* Segmentação de Dados
* Linha do Tempo
* Indicadores (KPIs)

## Principais Indicadores

* Faturamento Total
* Quantidade de Vendas
* Melhor Vendedor
* Participação por Região
* Faturamento por Região
* Faturamento por Produto
* Faturamento por Vendedor

## Ferramentas Utilizadas

* Microsoft Excel
* Tabelas Estruturadas
* Tabelas Dinâmicas
* Gráficos Dinâmicos
* Segmentação de Dados
* Linha do Tempo

## Habilidades Demonstradas

* Excel Avançado
* Construção de Dashboards
